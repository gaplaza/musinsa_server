# 프로젝트 소개
 대규모 트래픽 환경을 가정하여 부하 분산과 데이터 처리 효율성에 초점을 맞춘 무신사 클론 프로젝트입니다.
 특히 '블랙 프라이데이'와 같은 피크 타임에 특정 기능의 병목이 전체 시스템 장애로 번지지 않도록 DDD 기반 설계를 통해 도메인 간 독립성을 확보했습니다.
